# Screenshots

Drop the README's images here (referenced from the top-level `README.md`):

- **`web-ui.png`** — the local web UI with files loaded. Run `arc web`, drop in
  `examples/threads/`, and capture the left panel showing the keep/redundant
  badges and the timeline preview.
- **`timeline.png`** — a rendered timeline. Run
  `arc timeline examples/events.json -o timeline.html`, open it, and capture a
  tab with the SVG overview axis, filters, and a couple of expanded cards.

A short GIF (e.g. `web-ui.gif`) of the drag-drop → keep/delete → timeline flow
makes the strongest first impression if you can record one.

All capture material uses the **synthetic Voltera sample data**, so the images
are safe to publish.
