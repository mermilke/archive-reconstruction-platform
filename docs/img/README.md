# Screenshots

The README's images live here (referenced from the top-level `README.md`):

- **`demo.gif`** — the hero: a short screen recording of the local `arc web`
  flow. Run `arc web`, drop in an export folder (e.g. `examples/pdf_emails/`),
  show the keep/redundant verdict, open a thread to read it inline, then build a
  timeline. Recorded with ScreenToGif.
- **`timeline.png`** — a rendered timeline. Run
  `arc timeline examples/events.json -o timeline.html`, open it, and capture a
  tab with the SVG overview axis, filters, and a couple of expanded cards.

All capture material uses the **synthetic Voltera sample data**, so the images
are safe to publish.
