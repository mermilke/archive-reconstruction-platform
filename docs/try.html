Archive Reconstruction Platform

Branch-aware dedup, right in your browser. 100% in‑browser · no upload View on GitHub →

Got a folder full of exported email threads — the pile you get after archiving a mailbox or saving conversations one at a time? Drop them here and this tool tells you which files are redundant copies and which are the branches worth keeping — the ones that together still hold every message and attachment. Your files never leave this page; all the work happens in your browser.

No exports handy? to see it work.

Files & dedup verdict

    The tool only recommends which files are redundant — it never deletes anything. Click any file name to read it; redundant files open side-by-side with their keeper so you can see exactly what the keeper has that they don't.
