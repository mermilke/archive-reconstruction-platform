Archive Reconstruction Platform

Make sense of a folder full of overlapping email-thread exports — branch-aware deduplication and self-contained timelines, on the Python standard library.

When a conversation forks, the biggest file isn't always a superset. A smaller export can hold a reply — or an attachment — the big one never had. This tool compares the content of each export, not its size, so it flags a file as redundant only when every message and attachment it holds lives somewhere else. That branch-aware dedup is the whole idea.
